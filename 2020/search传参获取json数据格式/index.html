?(search)传参获取json数据格式

Written by 赵旭东 with ♥ on October 21, 2020 in
100 Words|Read in about 1 Min|本文总阅读量次

?(search)传参的情况下,拿到的this.props.location.search为"?a=10&b=20&b=30"的格式,但我们一般是用到json的数据格式是{a:“10”,b:“20”,c:“30”}这样的,所以我们要自己处理一下

1.原生js方法

let str=this.props.location.search.slice(1) ; //a=10&b=20&3=30
//"字符串截取,去掉'?'
let arr=str.split("&"); //["a=10", "b=20" , "c=30"]
//字符串split方法 把字符串变为数组,去掉&
let result={};
//声明一个空对象 一会保存json数据
arr.forEach(item=>{
	let temArr=item.split("=") ; // ["a","10"]
	result[temArr[0]]=temArr[1] //{"a":"10"}
})
//数组迭代方法forEach与split结合 去掉'='生成新的子数组, 并组合成json数据
consloe.log(result) //{"a":"10","b":"20","c":"30"}

2.node内置querystring方法

//首先需要引入node.js的方法
import querystring from "querystring";
let str=this.props.location.search.slice(1)
//同样先去掉?
let result = querystring.parse(str)
//然后直接使用方法
console.log(result) //{"a":"10","b":"20","c":"30"}

好像第二种比较简单,但作为初学者我认为还是得掌握原生js方法.
